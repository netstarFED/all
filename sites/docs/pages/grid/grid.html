参数说明

参数 说明 类型 必填 备注
columns 配置参数 array
data 数据配置 string
ui ui配置 string 
    {    
        columns:[...   //具体参数详见下面 ]
        data:{ ...  //具体参数详见下面}
        ui:{...  //具体参数详见下面}
    }

表格配置参数

参数说明
参数 说明 类型 必填 备注
field 不可重复字段名 string 字段id
title 标题 string {}
columnType 类型 string 表格列类型
editable 是否编辑 boolean 是否可以编辑
orderable 是否允许列排序 boolean
hidden 隐藏列 boolean
fieldLength 字段长度 string
rowColor 行颜色 object
total 表达式 object 举例：{{money}}-{{price}}
isTreeNode 是否有树节点 boolean
searchable 是否允许列搜索 boolean
isQuickQuery 是否支持快速查询 boolean
tooltip 提示信息 string
isTitle 是否作为标题 boolean 默认false
isValue 是否作为值 boolean 默认false
isDefaultSubdataText 默认字符串替换 boolean select/radio/checkbox时有效
editConfig 表单配置 object
getRowData 行数据 object
footer 类型 object sum（合计） average（平均值
formatHandler 列类型 object
isResizeWidth 是否支持某列拖拽 boolean 
    columns: [
        { ......//标题，类型等配置 举例如下},

        {  
            field: "num", 
            title: "数量",    
            columnType" "number",   
            variableType: "number",
            editable: true,
            isDefaultSubdataText:true,
            searchable:true,
            orderable:false,
            formatHandler: {
                type: "number",
                format: ",."
            },
            footer: {
                "type": "sum"//合计
            },
            editConfig: {
                "total": "{{money}}-{{price}}"
            },
            
        },
    ]

表格数据参数

参数说明
参数 说明 类型 必填 备注
idField 数据集中主键字段名，或不重复的字段名 object
dataSource 静态数据源 object
isPage 是否分页 boolean
isSearch 是否允许检索 boolean
isServerMode 是否允许本地检索 boolean 默认false
src 获取地址 string 通过ajax获得subdata
dataSrc 返回的值 object ajax返回数据的key值
type 类型 string ajax类型，GET/POST (默认GET)
data 传参 object ajax传参 关联字段配置（即获得表单中其它字段值作为传输参数）{}
contentType ajax的传值类型 string ajax的传值类型 ‘application/json’表示传值data转化成字符串 
    data:{
        idField:"",
        tableID:"//id名称"
        dataSourc:[,
            {
                num:"数字",
            }
        ]
    }

ui参数配置

参数说明
参数 说明 类型 必填 备注
isAutoSerial 是否序号 boolean
isCheckSelect 是否勾选 boolean
isHeader bolean
isPage 分页 boolean 是否可以分页
isAllowAdd 是否自动填充行 boolean
minPageLength 最小页面长度 number
isEditMode 是否可以编辑 boolean
pageLengthMenu 允许翻页长度 array [5, 10, 15, 20, 25, 30, 35, 40, 45, 50, 100, 200, 500, 1000, 2000],
pageLengthDefault 默认分页数量 number
toPageMenu 分页 number 默认只有一个分页
selectMode 选择 string 选择方式 single（单选）,multi（多选
styleExpress 样式表 string
topageDefaulc 第几条 string
isHaveEditDeleteBtn 是否有删除按钮 boolean 编辑模式下
tableRowBtns 表格行内按钮 function
height 高 number 默认200
ajaxSuccessHandler ajax完成回调 function
completeHandler 完成事件 function
pageChangeAfterHandler 翻页回调 function
pageLengthChangeHandler 改变条数回调 function
rowdbClickHandler 双击回调 function
originalRowsChangeHandler 原始行回调 function
selectedHandler 行选中 function
defaultSelectedIndex 默认选中第几条 number 举例：默认选中第2条数据（defaultSelectedIndex：1）
isUseMessageState 是否允许代办列 boolean
dragWidth 是否支持表格拖拽 boolean 默认true 不支持,

displayMode 默认表格展现形式 string table ， block
isOpenQuery 是否开启查询 boolean 默认 false
isOpenAdvanceQuery 是否开启高级查询 boolean 默认 false
isShowHead 是否显示标题头 boolean 默认 false
isReadStore 是否读取本地存储 boolean 默认 true 是
query 查询 object
isColumnCombine 是否支持列合并 boolean 当前只用于统计模板

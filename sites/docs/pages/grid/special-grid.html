grid

参数说明

参数 说明 类型 必填 备注
field id string 字段id
title 标题 string {}
columnType 类型 string 表格设置
editConfig 组件配置 object 组件配置参数
editable 编辑 boolean 是否可以编辑
formatHandler 格式化 object 

    {
        "field": "money",
        "title": "金额", //标题
        "columnType": "money",
        "editable": true,
        "variableType": "number",
        "editConfig": { 
            ...配置参数
        },
        formatHandler:{
            ...详见对应参数
        }
        "formatType": {
            ...详见以下参数            
        },
        "footer": {
        }
    }

参数 说明 类型 必填 备注
field 不可重复字段名 string 字段id

columnType 类型 string 表格设置

editConfig 组件配置 object 组件配置参数
editable 编辑 boolean 是否可以编辑

        },
        "formatType": {
            ...详见以下参数            
        },
        "footer": {
        }
    }

钱

参数 说明 类型 必填 备注
type 类型 金钱类型 money
data
format 格式化 string
places 小数位数 number 小数位数，默认2位
symbol 标识符 string 默认￥ $
thousand 千分分隔符 string 千分分隔符 默认','
decimal 小数点 string 小数点 默认显示小数点 （字符串替换，可以把小数点替换成其它字符。例：2.56（decimal：d）-->2d56） 
    columns: [{
        field: 'money1',
        title: '小数位数',
        columType: 'money',
        width: 200,
        editable: true,
        variableType:"number",
        formatHandler: {
            type: 'money',
            data: {
                format: {
                    places: 4,  //小数位数
                }
            }
        }
    }, {
        field: 'money2',
        title: '标识符',
        columType: 'money',
        width: 200,
        variableType:"number",
        editable: true,
        formatHandler: {
            type: 'money',
            data: {
                format: {
                    symbol: '$',//标识符
                }
            }
        }
    }, {
        field: 'money3',
        title: '千分分隔符',
        width: 200,
        columType: 'money',
        editable: true,
        variableType:"number",
        formatHandler: {
            type: 'money',
            data: {
                format: {
                    thousand: ',', //   千分分隔符
                }

            }
        },

    }, {
        field: 'money4',
        title: '小数点',
        width: 200,
        variableType:"number",
        columType: 'money',
        editable: true,
        formatHandler: {
            type: 'money',
            data: {
                format: {
                    decimal: "P",//小数点
                }

            }
        },

    }
    ],
    data: {...详见data参数配置},
    ui: {...详见ui参数配置}
    }

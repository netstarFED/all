businessSelect

参数说明

参数 说明 类型 必填 备注
id 字段名 string 组件id，实际生成的表单中名称为'form-'+id id不可重复
type 组件类型 string businessSelect
textField 显示字段 string 选项显示的字段(默认text)
valueField 保存值 string 选项value字段，即如果string类型（isObjectValue==false）组件的保存值 (默认value)
label 标签 string 组件的标签，未配置则不显示
rules 验证规则 string
value 显示的值 string 默认显示的值
disabled 只读 boolean 是否只读(默认flase)
inputWidth 宽 number 组件除去label的宽度与width不共存(默认100)
width 宽 string 组件整体宽度 *% 与inputWidth不共存
changeHandler 变更处理程序 function 组件值改变时触发 即value值发生改变时触发
hidden 隐藏 boolean 是否隐藏(默认flase)
subdata 数据 array 选项数组 具体配置见下方：subdata配置 数据类型[]
url 获取地址 string 通过ajax获得subdata
dataSrc 返回的值 string ajax返回数据的key值
type 类型 string ajax类型，GET/POST (默认GET)
data 传参 object ajax传参 关联字段配置（即获得表单中其它字段值作为传输参数）{}
contentType ajax的传值类型 string ajax的传值类型 ‘application/json’表示传值data转化成字符串
ajaxConfig ajax配置 object ajax配置
selectMode 选择 string 单选 single multi
formSource 组件模式 string 组件模式；直接调用组件时配置，表单配置时不需要配置直接通过表单配置设置了组件配置(默认from)
templateName 组件模式名字 string 组件模板名字；直接调用组件时配置，表单配置时不需要配置直接通过表单配置设置了组件配置(默认pc)
mindjetFieldPosition 是否在更多种显示 string 是否是更多中显示的数据(默认field)
listExpression 显示表达式 string 显示表达式
relationField 关联字段 string 举例：\"customerContacterId,addressTo\"
outputField 输出数据 string 弹出模板时需要
innerFields 初始化组件 string 组件赋值时value赋值
panelConfig 面板配置 string
idField id字段 string id字段(默认id)
assignExpres 表单赋值 object
isDistinct 是否有排重字段 boolean
distinctField 排重字段 string
valueExpression value表达式 string 通过表达式设置value值 {page.**}
placeholder placeholder string
gridPlaceholder 表格placeholder string 表格默认显示（业务组件：点击新增，其它：可输入）
source-ajax source-ajax object
getRowData getRowData object
getFormData getFormData object

显示的值

默认显示内容:日期组件value值必须是数字 
    {
        id: 'value',
        label: '显示 :',
        type: 'businessSelect',
        rules:"",
        textField: 'methodName',
        valueField: 'id',
        source:{
            url: getRootPath() + '/docs/pages/components/business-pages.html',
                // data: 'clickpage',
            data: [
                {aa:111},
                {bb:111},
                {cc:111},
            ],                             
        },
    }

只读

只读，不可编辑 
    {
        id: 'disabled',
        label: '只读 :',
        type: 'businessSelect',
        value: "2",
        disabled: true,//只读
        textField: 'name',
        valueField: 'id',
        subdata: [
            {
                id: '1',
                name: '男',
                state: "none",
                disabled: "true",
            },
            {
                id: '2',
                name: '女'
            },
            {
                id: '3',
                name: '其它'
            },
        ], 
    }

单选多选

单选single 多选multi 
    {
        id: 'businessselectMode',
        label: '只读 :',
        type: 'businessSelect',
        value: "1",
        textField: 'name',
        valueField: 'id',
        subdata: [{
                id: '1',

            },
            {
                id: '2',
                name: '女',
            },
            {
                id: '3',
                name: '其它',
            }
        ], 
        businessselectMode:"multi",   //单选single  多选multi              
    }

获取数据1

用subdata获取数据 
    {
        id: 'subdata',
        label: 'subdata传出:',
        type: 'businessSelect',
        value: "1",
        textField: 'name',
        valueField: 'id',
        subdata:[               //通过subdata获取值 
            {id:'1', name:'男'},
            {id:'2', name:'女'},
            {id:'3', name:'其它'}
        ]
    }

获取数据2

用ajax获取数据 
        {
            id: 'ajax',
            label: 'ajax传出:',
            type: 'businessSelect',
            value: "1",
            textField: 'name',
            valueField: 'id',
            ajaxConfig:{
                url: '/docs/data/businessselect/radio.json' ,
                type: 'GET',
                contentType: 'application/x-www-form-urlencoded', 
                dataSrc: 'rows',
            },        
        }

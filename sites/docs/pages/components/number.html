参数说明

参数 说明 类型 必填 备注
id 字段名 string 组件id，实际生成的表单中名称为'form-'+id id不可重复
type 组件类型 string number
label 标签 string 组件的标签，未配置则不显示
rules 验证规则 string
value 显示的值 string 默认显示的值
decimalDigit 小数点后几位 number 小数点后几位
isShowCalculator 是否显示计算器 boolean 默认是
disabled 只读 boolean 是否只读(默认flase)
inputWidth 宽 number 组件除去label的宽度 与width不共存(默认100)
width 宽 string 组件整体宽度 *% 与inputWidth不共存
changeHandler 变更处理程序 function 组件值改变时触发 即value值发生改变时触发
hidden 隐藏 boolean 是否隐藏(默认flase)
formSource 组件模式 string 组件模式；直接调用组件时配置，表单配置时不需要配置直接通过表单配置设置了组件配置(默认from)
templateName 组件模式名字 string 组件模板名字；直接调用组件时配置，表单配置时不需要配置直接通过表单配置设置了组件配置(默认pc)
mindjetFieldPosition 是否在更多种显示 string 是否是更多中显示的数据(默认field)
isShowCalculator 是否显示计算器 string 默认true
isDistinct 是否有排重字段 boolean
total 表达式 boolean 举例：{{refFeeTotal}}-{{discount}}
distinctField 排重字段 string
valueExpression value表达式 string 通过表达式设置value值 {page.**}
placeholder placeholder string
gridPlaceholder 表格placeholder string 表格默认显示（业务组件：点击新增，其它：可输入）

显示的值

默认显示内容:日期组件value值必须是数字 
    {
        id: 'value',
        label: '显示 :',
        acts: "label",
        type: 'number',
        value: '24',       //默认显示24
    }

只读

只读，不可编辑 
    {
        id: 'disabled',
        label: '只读 :',
        type: 'number',
        value: "",
        disabled: true,//只读模式
    }

样式宽

宽 
    {
                
        id: 'width',
        label: 'width宽:',
        type: 'number',
        value: "",
        iwidth:300,    //宽

    }

小数点后几位

    {
        isHasClose: false, 
        id: 'isHasClose',
        label: '小数:',
        type: 'number',
        value: "24",
        width: 500,
        decimalDigit:2,//小数后面有两位
    }

是否显示计算机

    {
        id: 'isshowcalculator',
        label: '是否显示计算机:',
        type: 'number',
        value: "24",
        width: 500,
        isshowcalculator: false, //小数后面有两位

    }

验证1

验证输入内容是否正确注：ajax验证时remoteAjax必填 
    {
        id: 'width',
        label: '验证:',
        type: 'text',
        value: '',
        rules: 'email', //  /mobile/fax/isphone/postalcode/bankno/Icd/positiveInteger
                        //  /nonnegativeInteger/integer/min/max/positive/negative
                        //  /range/precision/email
    }

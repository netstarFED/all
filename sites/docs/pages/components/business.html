参数说明

参数 说明 类型 必填 备注
id 字段名 string 组件id，实际生成的表单中名称为'form-'+id id不可重复
source 获得 object 弹框调取的地址 参数：url：‘’ 默认{}
searth 路径 object 回车搜索配置 参数：url：‘’；dataSrc：‘’；默认{}
type 组件类型 string business
textField 显示字段 string 选项显示的字段(默认text)
dialogTitle 弹框标题 string 弹框标题(默认默认弹框标题)
infoBtnName 基本信息 string 基本信息
label 标签 string 组件的标签，未配置则不显示
rules 验证规则 string 当前组件仅支持必填验证：required
value 显示的值 string 默认显示的值组件vo是一个对象{}
disabled 只读 boolean 是否只读(默认flase)
inputWidth 宽 number 组件除去label的宽度 与width不共存(默认100)
idField 主键字段 string
width 宽 string 组件整体宽度 *% 与inputWidth不共存
changeHandler 变更处理程序 function 组件值改变时触发 即value值发生改变时触发
hidden 隐藏 boolean 是否隐藏(默认flase)
isInputText 是否支持输入 boolean (默认false)支持输入
isReadOutputFields 是否配置outputFields string 是否配置outputFields(false)
defaultPageIndex 默认显示页码 number 默认0，
isOutputString 是否输出字符串 string 默认false
assignExpres vo赋值 object 通过业务组件返回值给表单赋值（vo赋值）
innerFields 初始化组件 object 组件赋值时value赋值举例
isAsync 是否异步加载 object
displayMode displayMode object tree,blockList
relationField 关联字段 string
outputFields 获取值 object
validateParams 参数验证（提示语） object 只验证指定参数例{"pageData":"msg":"提示内容","rules":"{字段名+字段名}<=SUM{字段名}"}
parameterFormat 参数格式化 object 举例：{"id":"{xxx}"}
showExpression 表单显示表达式 object
voField vo字段 string
selectedKey 选中数据名称 string
isDistinct 是否有排重字段 boolean
distinctField 排重字段 string
valueExpression value表达式 string 通过表达式设置value值 {page.**}
isDeleteObjectState 是否删除ObjectState boolean
formatValueData 操作数据 object 将页面面数据传递到另一个页面
defaultSearchData 查看默认数据 number 支持tab页，默认第一个
isNeedRowData 是否添加行 boolean 默认true,配置false时只给当前字段赋值
placeholder placeholder string
gridPlaceholder 表格placeholder string 表格默认显示（业务组件：点击新增，其它：可输入）
getRowData 表格行赋值 object
assignExpres vo赋值 object 拿当前行数据给表单赋值
getFormData 表单赋值 object 拿ajax返回数据给表单赋值
  • *业务组件当前弹出模板只支持基本业务模板（businessDataBase）
  • * 关于配置业务组件常见错误
  •     1.value赋值错误，注意idField是否是正确，或者字段是否与idField重复
  •     2.value显示错误，注意显示字段（textField）是否配置正确
  •     3.保存不上时，注意初始化组件，与获取值（outputFields，innerFields）是否配置正确具体配置请参照文档
  •     4.当表格内既有业务组件又有其它组件时，业务组件必须在第一个选择。如果需要第一个选择其他组件可以配置isNeedRowData
  •     当isNeedRowData为false时只给当前字段赋值，不会进行行赋值
  • *关于outputFields与innerFields 一个是输出数据，一个是输入数据（输出数据时要确定当前值后台可以接）
  • *getFormData与assignExpres没有关系（注意不可同时配置）

        id: 'value',
        label: '默认显示',
        dialogTitle:"业务组件弹框1 , 业务组件弹框2",
        type: 'business',
        textField: 'cityname',
        idField: 'id',
        value: {
            cityname : '可点击'
        },
        source: {
            url: getRootPath() + '/docs/pages/components/business-pages.html',

            ],
        },

            dataSrc: 'rows',
        },
    }

多赋值

多个赋值 
    {
        id: 'value',
        label: '默认显示',
        dialogTitle:"业务组件弹框1 , 业务组件弹框2",
        type: 'business',
        textField: 'cityname',
        idField: 'id',
        source: {
            url: getRootPath() + '/docs/pages/components/business-pages.html,' + getRootPath() + '/docs/pages/components/business-pages2.html',
            data: [
                {aa:111},
                {bb:111},
                {cc:111},
            ],
        },
        search: {
            url: getRootPath() + '/assets/json/newcomponent/search.json',
            dataSrc: 'rows',
        },
        selectMode: 'single',
        outputFields : {
            tableName : "{cityname}",
            xx:"{itemCateName}"
        },
        innerFields : {
            tableName : "{tableName}",
            itemCateName : "{itemCateName}",
            cityname : "{cityname}",
            id : "{value}",
        },
        showExpression : '{{itemCateName}},{{cityname}}',
        assignExpres : {
            area : "{itemCateName}",
            'posi-tion' : "{itemName}",
            'day2' : "{orderId}",
        },

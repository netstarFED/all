参数说明

参数 说明 类型 必填 备注
id 字段名 string 组件id，实际生成的表单中名称为'form-'+id id不可重复
type 组件类型 string text
placeholder 提示信息 string
tooltips 提示信息 string
isInputMask InputMask组件 boolean 当值为true时 ，format必填
format 格式化 string
readonly 只读 boolean
isOnKeydown 是否焦点切换 boolean 是否设置快捷键切换焦点
rules 规则 string
formSource 模式 string halfScreen (半屏) fullScreen(全屏) inlineScreen（行内）staticData（功能）

staticData

参数 说明 类型 备注
formatFunc 标签 string
label 标签 string
title 标签 string radio
formlabel 表单标签 string
tel 电话号 string
link 一个普通连接 boolean
block 一个块元素 string
baidu 一个普通连接 string
baiduMapByName 百度地图 string
doubt 问号 string
icons 一个图标链接 string
date 一个日期 string
datetime 一个时间 string
qqMapByName 腾讯地图 boolean

参数说明

参数 说明 类型 必填 备注
id 字段名 string 组件id，实际生成的表单中名称为'form-'+id id不可重复
type 组件类型 string date
placeholder 提示信息 string
tooltips 提示信息 string
isInputMask InputMask组件 boolean 当值为true时 ，format必填
daysOfWeekDisabled 只读 string 字符串或数组格式 根据周几设置只读 默认''即不设置只读
format 日期格式 string 日期格式 yyy-mm-dd (默认：nsVals.default.dateFormat)
startView 开始视图 number 开始视图 0(日)/1(月->日)/2(年->月->日)(默认0)
addvalue 添加按钮 object 日期底部按钮 配置 {value:''，id:''}({})
readonly 只读 boolean
rules 规则 string required 必填
formSource 模式 string halfScreen (半屏) fullScreen(全屏) inlineScreen（行内）staticData（功能）

staticData

formatFunc 标签 string

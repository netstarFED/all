wxtPrint

通用参数说明

参数 说明 类型 备注
defaultMode 类型 string 弹框addInfoDidalog
functionClass 方法类型 string modal/list
url 地址 string
type 传参类型 string ajax类型，GET/POST (默认GET)
contentType ajax的传值类型 string ajax的传值类型 ‘application/json’表示传值data转化成字符串
text 按钮文字 string
keyField 关键字 string
title 弹框标题 string
listName 列表名字 string
isMainDbAction 是否双击事件 boolean 是否将当前方法定义为表格行双击时执行的方法
isInlineBtn 是否行内按钮 boolean
isMobileInlineBtn 是否手机行内按钮 boolean
disabledExpression 禁用表达式 object {1:'disabled'}
formatValueData 界面格式化转参 string
parameterFormat 参数格式化 object 举例：{"id":"{xxx}"}
parameterFormatType 参数格式化类型 string 所定义的参数是否会覆盖默认参数
targetField 目标字段 string 目标字段 （keyField）
btnType 按钮类型 string print preview printAjax previewAjax
dropdownSubdata dropdownSubdata string
dropSelectMode dropSelectMode string checkbox single
disabledByWorkflow 工作流按钮只读 boolean

通用参数说明

参数 说明 类型 备注
defaultMode 类型 string business业务组件
functionClass 方法类型 string modal/list
url 地址 string
type 传参类型 string ajax类型，GET/POST (默认GET)
contentType ajax的传值类型 string ajax的传值类型 ‘application/json’表示传值data转化成字符串
text 按钮文字 string 按钮中所显示的内容
dialogTitle 弹框标题 string
selectMode 选择模式 string single单选 checkbox多选 noSelect不能选
operatorMode 操作更多 string add 新增 edit 编辑
getPanelDataAjax 获取数据ajax object
requestSource 请求参数来源 string selected选中行 checkbox勾选行 thisVo当前数据 none无
isHaveSaveAndAdd 是否保存新增 boolean
isUseAjaxByCopyAdd 使用ajax复制 boolean
parameterFormat 参数格式化 object> 举例：{"id":"{xxx}"}
validateParams 参数验证（提示语） object 只验证指定参数例{"pageData":"msg":"提示内容","rules":"{字段名+字段名}<=SUM{字段名}"}
matrixVariable 矩阵传值 object
isSendPageParams 是否发送页面参数 boolean
shortcutKey 快捷键 string
isKeepSelected 是否行选中 boolean
targetField 目标字段 string 目标字段 （keyField）

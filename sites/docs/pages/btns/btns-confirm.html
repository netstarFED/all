confirm

通用参数说明

参数 说明 类型 备注
defaultMode 类型 string confirm（确定按钮）
functionClass 方法类型 string modal/list
isCloseWindow 是否关闭 boolean
type 传参类型 string ajax类型，GET/POST (默认GET)
dataFormat 格式化参数 string 两个值 id object
dataSrc 返回的值 string 两个值 data/rows
text 按钮文字 string
title 确认弹框标题 string
ajaxData data入参值 object
requestSource 发送形式 string selected（选中行）checkbox（勾选行）thisVo（当前数据）none（无）
isMainDbAction 是否执行双击事件 boolean
isInlineBtn 是否行内按钮 boolean
isMobileInlineBtn 是否手机行内按钮 boolean
disabledExpression 禁用表达式 string {1:'disabled'}
parameterFormat 参数格式化 string 举例：{"id":"{xxx}"}
validateParams 参数验证（提示语） object 只验证指定参数例{"pageData":"msg":"提示内容","rules":"{字段名+字段名}<=SUM{字段名}"}
successMsg 成功提示语 string
successOperate 完成后页面动作 string refresh(刷新) delete(删除) edit(编辑) add(添加) clearOrClose(清空或关闭 当前只在lims使用)
matrixVariable 矩阵传值 object 具体形式以API文档为主
isSendPageParams 是否发送页面参数 boolean 参数发送时是否包含页面来源参数
shortcutKey 快捷键 string "举例:ctrl+e"
isKeepSelected 是否行选中 boolean
targetField 目标字段 string 目标字段 （keyField）

templatePrint

通用参数说明

参数 说明 类型 备注
defaultMode 类型 string templatePrint
functionClass 类名 string modal/list
text 按钮文字 sting 按钮中所显示的内容
shortcutKey 快捷键 string 举例:ctrl+e
isInlineBtn 是否行内按钮 boolean 默认false
textField 显示字段 string 下拉框显示字段
valueField 保存值 string 服务器接受到的返回值
webSocketUrl 自定义参数 string
callbackAjax 打开成功后的回调方法 string
btnType 按钮类型 string print preview printAjax previewAjax
requestSource 请求参数来源 string selected选中行 checkbox勾选行 thisvo当前数据 none无
validateParams 参数验证公式 string {'customerId':客户必填'},目前只支持是否存在的验证
isSendPageParams 是否发送页面参数 boolean 参数发送时是否包含页面来源参数
isSetValueToSourcePage 是否对来源页面赋值 boolean 是否使用API返回的数据对来源页面赋值
disabledExpression 禁用表达式 object 1：disabled
matrixVariable 矩阵传参 object 具体形式以API文档为主
parameterFormat 参数格式化 object 举例：{"id":"{xxx}"}
parameterFormatType 参数格式化类型 string 所定义的参数是否会覆盖默认参数
targetField 目标字段 string 目标字段 （keyField）

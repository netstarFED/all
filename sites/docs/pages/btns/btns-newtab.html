newtab

通用参数说明

>
参数 说明 类型 备注
defaultMode 类型 string newtab(打开新的tab页)
functionClass 类名 string modal/list
isCloseWindow 是否打开页面 boolean
type 传参类型 string ajax类型，GET/POST (默认GET)
dataSrc 返回的值 string 两个值 data/rows
text 文本 string
title 标题 string
url 地址 string
requestSource 发送形式 string selected（选中行）checkbox（勾选行）thisVo（当前数据）none（无）
isMainDbAction 是否执行双击事件 boolean
isAlwaysNewTab 是否打开新的tab页 boolean
isEditMode 编辑状态 boolean
isInlineBtn 是否行内按钮 boolean
isMobileInlineBtn 是否手机行内按钮 boolean
disabledExpression 禁用表达式 object {1:'disabled'}
parameterFormat 参数格式化 object 举例：{"id":"{xxx}"}
validateParams 参数验证（提示语） object 只验证指定参数例{"pageData":"msg":"提示内容","rules":"{字段名+字段名}<=SUM{字段名}"}
matrixVariable 矩阵传值 object
isSendPageParams 是否发送页面参数 boolean
formatValueData 界面格式化转参 string
shortcutKey 快捷键 string
isKeepSelected 是否行选中 boolean
targetField 目标字段 string 目标字段 （keyField）

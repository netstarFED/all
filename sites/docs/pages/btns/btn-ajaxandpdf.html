ajaxandpdf

通用参数说明

参数 说明 类型 备注
defaultMode 类型 string 弹框ajax打印
functionClass 方法类型 string modal/list
type 传参类型 string ajax类型，GET/POST (默认GET)
dataSrc 返回的值 string ajax返回数据的key值
ajaxData ajax传参 object
fileField fileField string {\"fileId\":\"{fileId}\"}"
requestSource 请求参数来源 string selected选中行 checkbox勾选行 thisvo当前数据 无none
targetField 目标字段 string 目标字段 （keyField）

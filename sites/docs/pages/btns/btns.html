btns

参数说明

参数 说明 类型 必填 备注
id 唯一标识 string 唯一名称
position 位置 string 'text-left'
btns 按钮配置 array 具体配置详见下方
text 文本 string 具体配置详见下方
dropdownType 按钮组 string memoryDropdown
disabled 只读 boolean 具体配置详见下方
handler 方法 function handler：function(){}

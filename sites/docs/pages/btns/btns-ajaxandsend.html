参数 说明 类型 备注
defaultMode 类型 string ajaxAndSend 打印用的（调用打印之前先发送ajax获取需要打印用到的参数）
functionClass 方法类型 string
text 按钮文字 string 按钮中所显示的内容
webSocketBody 自定义参数 string {"command":"批量打印PDF","business":[{fileId:"1318695811176988773",copies:1}]}
callbackAjax 打印成功之后的回调方法 object
requestSource 请求参数来源 string selected选中行 checkbox勾选行 thisvo当前数据 无none
isSetValueToSourcePage 是否对来源页面赋值 boolean 是否使用API返回的数据对来源页面赋值
targetField 目标字段 string 目标字段 （keyField）

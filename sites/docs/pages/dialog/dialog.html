参数说明

参数 说明 类型 必填 备注
id 唯一标识 string 唯一名称
tempalte 组件模式 string 默认"PC"
height 高 number 可以使用百分比
width 宽 number 可以使用百分比
isAllowDrag 拖拽 boolean 是否允许拖拽
plusClass 添加样式 string 添加样式
isStore 是否记录弹框 boolean 是否记录弹框位置和大小信息 按照百分比计算
showHandler 显示前回调 function
shownHandler 显示后回调 function
mousedownHandler 按下鼠标后回调 function
mousemoveHandler 移动鼠标回调 function
hideHandler 隐藏弹框前回调 function
hiddenHandler 隐藏弹框后回调 function

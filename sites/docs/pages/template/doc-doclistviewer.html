单据详情模板

参数说明

参数 说明 类型 必填 备注
package 包名 string 不可重复
template 模板 string 单据详情列表docListViewer
title 标题 string
mode 显示模式 string expandlist
components 组成部分 array []
getValueAjax getValueAjax object
readonly 只读 boolean 默认false
isShowTitle 是否显示标题 boolean
isFormHidden 表单是否隐藏 boolean 默认false
draftBox 草稿箱 object fields(草稿箱字段） isUse（是否显示草稿箱false）isUseSave（是否显示草稿箱保存true）closeOrClear（草稿箱关闭或清楚true）
isSaveToTemplate 是否保存为模板 boolean 默认false
plusClass 样式 string
pushMessage 推送消息 string
pageExpression 页面表达式 object
isUseBtnPanelManager 是否使用按钮面板 string 默认false
params 参数 string 特殊参数 

      $(function(){
          var docDocListViewer={
              package:'',
              template:'',
              title:'',
              components:[
              {
                type:'tree',
                ......//具体配置详见树文档
              }，{
                type:'list',
                ......//具体配置详见表格文档
              }
            ]
        }
        NetstarTemplate.init(docDocListViewer);
      })

单据详情列表页面一

单表格 只输出一个list或者blockList的表格(grid)

单据详情列表页面二

纵向展示 两个list 并列展示,适用于上表格+下tab中多个list表格 （vertical）

单据详情列表页面三

两个list加一个blockList 其中一个list为主数据，另一个list和blockList为同级结构的数据

1.在配置两个表格时，需要先定义主表keyField："root"

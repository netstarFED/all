tree

参数说明

参数 说明 类型 必填 备注
type 类型 string 当前为tree
ajax 传入数据 object 传入的数据
addAjax 新增ajax object 新增时获取的数据
moveAjax 移动Ajax object 移动时获取的数据
editAjax 修改Ajax object 修改时获取的数据
deleteAjax 删除Ajax object 删除时获取的数据
height 树高度Ajax number 树的高度（外边框）
width 树宽度 number 树的宽度（外边框）
textField 选项显示的字段, string 选项显示的字段 （默认name）
parentField 父节点名称 string 父节点的id
childField 树数据子级名 string 树数据子级名
fromField 新增时发送字段 string 新增时发送的id
title 标题 string 树的标题
plusClass 样式 object 单独样式
level 打开层级 number 默认全部关闭
isSearch 是否允许检索 boolean 是否允许检索
isMultiple 是否多选 boolean 是否多选
readonly 是否只读 boolean 是否只读（只读可点击）
maxlength 节点name长度 number 节点name长度
unfieldId 子级id string 子级名称的id
unfieldText 子级名 string 新增子级名
allfieldText 父级名称 string 父级名称
isClickRoot 是否选中根节点 boolean 单选

        type: "tree",  //类型

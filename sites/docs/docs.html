参数 类型 说明 默认值 备注
1 id string 当前类型元素的id 无，必填 实际生成的表单中名称为'form-'+id
2 label string 当前类型元素的文本 无
3 type string 当前类型元素的类型 无，必填 日期时间类型的type必须是select
4 rules string 必填验证 无 日期时间的类型必填验证
5 coloum Number 所占的列宽 无 值为1-12的数值
6 value string 默认显示的值 无
7 textField string 文本属性的字段值
8 valueField string 文本值属性的字段值
9 disabled boolean 是否不可编辑 false
10 subdate Array 属性值拼接，如果是ajax请求方式此项忽略

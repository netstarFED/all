PDF插件 nsUI.pdfview
  • 作用：以全屏弹出框形式打开pdf
  • 第三方插件地址：assets/js/pdf/pdf.js
  • nsUI.pdfView文件地址：assets/js/ui/nsui-pdfview.js
  • 该组件需要 nsFrame.contentDialog组件，该组件位于common/nsframe.js
使用方式：nsUI.pdfView.show({url:url});
参数 类型 说明 默认值 备注
1 url string 请求的url链接 无 必填
2 pageNumber number 打开第几页 1
3 isDownload boolean 是否允许下载 false
4 isPrint boolean 是否允许打印 false 
	//直接打开
	nsUI.pdfView.show({
		url:'../../image/wps-w.pdf',
	});
	
	//带按钮
	url = '../../image/1.pdf';
	nsUI.pdfView.show({
		isDownload:true,//下载
		isPrint:true, 	//打印
		url:url
	});

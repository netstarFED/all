选择项目组件说明

项目组件按钮生成调用方法：nsUI.projectSelect.init(config)

该方法调用后，第二次点击会重现之前的操作。
如果需要打开时重新初始化，需要先进行清空操作，请使用下面的清除方法

调用方法：nsUI.projectSelect.clear()

navJson参数说明

参数 类型 说明 默认值 备注
1 id string 生成项目组件按钮的id 无,必填
2 btns [] 要生成的按钮数组 无,必填 btns为数组，但数组中只能包含一个按钮，不支持多个
3 btns text string 标题，要显示的按钮标题 无,必填
4 btns handler function 调用项目组件的方法 无,必填

navJson使用示例：

	var navJson = {
		id: "project-select-nav",
		btns:[
			[
				{
					text: 		'项目选择',
					handler: 	test,
					isReturn: 	true
				}
			]
		]
	}
	controlPlane.formNavInit(navJson);
	function test(ev){
		nsUI.projectSelect.init(config)
	}

调用项目组件方法：nsUI.projectSelect.init(config)

config参数说明

参数 类型 说明 默认值 备注
1 $btn container 当前点击的按钮元素 $(ev.target) 在传该参数值时，必须传送该默认值
2 selected Array 已选择的列表的默认值 无
3 classWidth number string 宽配置 无 支持两种类型的，即宽度可以输入为数字类型，也可配置字符串类型
4 classAjax object 所有分类的配置 无
5 classAjax url string 分类请求的链接 无,必填
6 classAjax type string 分类请求的方式 post 两种支持方式post和get,默认请求数据方式是post
7 classAjax data object 分类请求的参数 空对象
8 classAjax dataSrc string 分类请求的数据源参 rows 返回的数据源参，默认读取的是rows
9 classAjax textField string 分类配置的文本字段 无，必填
10 classAjax valueField string 分类配置的id字段 无，必填
11 classAjax value object 分类配置的默认分类,value值和文本值必须和配置的valueField和textField字段一致 无 举例：{valueField:'',textField:''}
12 listAjax object 所有列表的配置 无，必填
13 listAjax url string 列表请求的链接 无,必填
14 listAjax type string 列表请求的方式 post 两种支持方式post和get,默认请求数据方式是post
15 listAjax data object 列表请求的参数 空对象
16 listAjax dataSrc string 列表请求的数据源参 rows 返回的数据源参，默认读取的是rows
17 listAjax textField string 列表配置的文本字段 无，必填
18 listAjax valueField string 列表配置的id字段 无，必填
19 listAjax classField string 列表配置的分类字段 无，必填
20 listAjax pyField string 列表配置的拼音字段 无，必填
21 listAjax wbField string 列表配置的五笔字段 无，必填
22 listAjax childField string 子元素字段 无，必填
23 listAjax selectField string 是否选中 无，必填
24 confirmHandler function 返回参数 无，必填
25 pageLength number 显示条数 默认显示10条
26 treeAjax object 配置下拉树 无 刷新左侧列表
27 treeAjax url string 下拉树请求的链接 无，必填
28 treeAjax treeType string 下拉树请求的方式
29 treeAjax data object 下拉树请求的参数
30 treeAjax dataSrc string 下拉树请求的数据源 rows
31 treeAjax textField string 下拉树配置的文本字段
32 treeAjax valueField string 下拉树配置的id字段
33 treeAjax ajaxDataField array，string 传送的数据 无 传多个参数['',''](数组);传一个值''(字符串)
34 treeAjax value string 默认值 无
35 treeVisible boolean 是否显示下拉树 true,显示
36 treeWidth number 下拉树的宽度
37 searchVisible boolean 是否显示搜索项 true,显示

config使用示例：

	var config = {
			$btn: 		$(ev.target),
			selected: 	'["000004","000006","1000013"]',  //选中的
			classWidth:  	 200,
			classAjax: 	{
				url: 		 getRootPath() + '/assets/json/project-class.json',
				type: 		 'GET',
				data: 		 {id:1},
				dataSrc: 	 'rows',
				textField: 	 'text',
				valueField:  	'dictId',
				value:{
					dictId:'0305',
					text:'现场(0305)',
				}
			},
			listAjax: 	{
				url: 		getRootPath() + '/assets/json/project-list.json',
				type: 		'GET',
				data: 		{id:1},
				dataSrc: 	'row',
				textField: 	'itemName',
				valueField: 	'itemCode',
				classField: 	'itemCateId1',
				pyField: 	'py',
				wbField: 	'wb',
				childField:'children',
				selectField:'isselect',
			},
			confirmHandler: confirmHandler,
	}
	nsUI.projectSelect.init(config);
